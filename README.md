## 项目简介
C++实现位图处理库

## 实现功能
* **平移** 
* **缩放** 
* **镜像** 
* **旋转** 
* **灰度** 
* **二值** 
* **平滑**
* **锐化**
* **腐蚀** 
* **膨胀** 
* **.......** 

## 项目后续
支持Png/Jpg等常用图片格式
